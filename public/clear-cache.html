🔄 Clear Cache - Bradley Health

✅ Cache Clearing Instructions

Follow these steps to see the new heart icons:

🖥️ Desktop Browsers:

  1. Chrome/Edge: Press Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
  2. Firefox: Press Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
  3. Safari: Press Cmd+Option+R (Mac)
  4. Alternative: Open Developer Tools (F12) → Right-click refresh button → "Empty Cache and Hard Reload"

📱 Mobile Browsers:

  1. Chrome: Settings → Privacy and Security → Clear browsing data → Cached images and files
  2. Safari: Settings → Safari → Clear History and Website Data
  3. Alternative: Close and reopen the browser app completely

🧪 Test New Icons:

💡 Pro Tip:

If you still see "W" icons, try opening the app in an incognito/private window to bypass all caches!
